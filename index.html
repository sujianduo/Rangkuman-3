1. 他 是 我 爸 爸 。 ( tā shì wǒ bà ba )
Dia adalah ayah saya.

2. 我 们 踢 足 球 。 ( wǒ men tī zú qiú )
Kami bermain sepakbola.

3. 我 做 作 业 。 ( wǒ zuò zuò yè )
Saya mengerjakan PR/tugas.

4. 哥 哥 打 篮 球 。 ( ɡē ɡe dǎ lán qiú )
Kakak laki-laki bermain bola basket.

5. 我 的 一 天 。 ( wǒ de yī tiān )
Kegiatan saya dalam sehari.
